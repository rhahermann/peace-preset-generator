# Presets to Create

Add music styles below, one per line. When ready, ask Claude to generate them.

## Format
```
[Genre/Style Name] - [Optional: variation] @ [Optional: Headphone Model]
```

If no `@ [Model]` is specified, the active headphone from CONFIG.md will be used.

## Examples
```
Rock - Heavy Metal
Electronic - Deep House @ Sennheiser HD 4.40 BT
Jazz - Smooth Evening @ Sony WH-1000XM5
Ambient
```
